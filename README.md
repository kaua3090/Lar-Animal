Lar Animal

O Lar Animal foi desenvolvido como Projeto Integrador do Senac Itaquera, com o objetivo de criar uma plataforma web para uma ONG de adoção de animais. O sistema foi projetado para facilitar o processo de adoção responsável, ampliar a visibilidade dos animais disponíveis para um novo lar e otimizar o gerenciamento das informações da instituição.

A aplicação conta com funcionalidades voltadas tanto para os usuários interessados em adoção quanto para a administração da ONG, proporcionando uma experiência intuitiva e eficiente na divulgação e acompanhamento dos animais cadastrados.

Tecnologias Utilizadas
HTML5 para a estruturação das páginas;
CSS3 para estilização e responsividade da interface;
JavaScript para interatividade e dinamismo da aplicação;
PHP para o desenvolvimento do back-end e integração com o banco de dados;
MySQL para armazenamento e gerenciamento dos dados.
Bibliotecas e Ferramentas

Durante o desenvolvimento foram utilizadas bibliotecas JavaScript que contribuíram para uma experiência mais moderna e interativa:

Swiper.js para criação de carrosséis e sliders responsivos;
Chart.js para exibição de gráficos e visualização de dados administrativos.

O projeto proporcionou a aplicação prática de conceitos de desenvolvimento web full stack, integração com banco de dados, responsividade, experiência do usuário e organização de sistemas voltados para causas sociais.
